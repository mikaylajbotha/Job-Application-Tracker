# Job Application Tracker

A full-featured, production-quality job application tracker built with **pure HTML, CSS, and vanilla JavaScript** — no frameworks, no build step.

Built in December 2025 while applying to software engineering roles in Austin, TX.

### Features
- Kanban board with native drag & drop
- Real-time dashboard + rejection rate tracking
- Chart.js pie & bar charts
- CSV + JSON export/import
- Confetti explosion when you get an Offer or Accepted
- Ghosting detection (14+ days no update → warning badge)
- Deadline tracking + one-click Google Calendar .ics export
- Dark mode ready (CSS variables + toggle)
- Search, filter, sort, priority & dream company flags
- Auto-archive old rejected applications
- Fully responsive + print-friendly
- 100% client-side with localStorage persistence

### Tech Stack
- HTML5
- Vanilla JavaScript (ES6+)
- CSS Custom Properties (light/dark mode)
- Bootstrap 5 (only modal + responsive grid)
- Chart.js + canvas-confetti
- Zero npm dependencies

### How to Run
1. Clone or download the repo
2. Open `index.html` in any browser — works offline instantly

### Why I Built This
Excel and Notion kept breaking on my phone.  
So I built the exact tool I wished existed — clean, fast, and actually fun to use.

Keyboard shortcuts:  
`N` → Add new application
